# 배달의 민족 프로젝트

### :wrench: 기능설명

+ Naver Map APi를 이용하여 지도를 띄운다.
+ 내 위치를 받아와 화면을 이동해준다.
+ Mock API에서 예약가능 숙소 목록을 받아와서 지도에 표시한다.
+ BottomSheetView를 활용하여 예약 가능 숙소목록을 볼 수 있다.
+ ViewPager2를 활용하여 현재 보고 있는 숙소로 지도가 움직이며 간단한 섬네일 정보를 볼 수 있다.
+ 숙소 버튼을 눌러 숙소의 장보를 앱 외부로 공유 가능하다.
